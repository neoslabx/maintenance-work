== Description ==

Add a maintenance page to your website that lets visitors know your page is down for maintenance while allowing you to work on the background. The users of your website having admin privileges gets full access to the page including the front end.

== Installation ==

1. Upload "maintenance-work" folder to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Click the "Maintenance Mode" link located under your "Settings" section to configure the plugin options.
